Testing OpenConext

Integration testing is performed from within the OpenConext-VM. You need to log in to the OpenConext-VM and perform the following steps to start with integration testing:

  • run 'startx' this will start a graphical environment with a terminal and a browser
  • Import the Certificate Authority into your browser (the browser inside the VM)
  • Add the Selenium IDE to your browser
  • Start a new private browsing window
  • Start selenium IDE
  • load test suite and run
